Catch buses & trains on time with
Bed2Transit

Monitors multiple bus stops and BART trains to save you time in deciding
which stops you can arrive to on time from your bed.

Notes
  • Bed2Transit can only monitor buses that run under AC Transit and trains from BART.
  • Data for AC Transit buses is provided by 511.org, http://www.511.org/
Easy to use
  1. Select a bus and bus stop or BART station
  2. Set time it takes you to walk there
  3. Set time it takes you to get ready
  4. Add to list of monitored stops
